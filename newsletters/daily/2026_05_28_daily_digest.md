<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Snowflake beats and raises, lands a $6B AWS Graviton + AI chip deal, and buys Natoma — SNOW +~37% AH.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>

</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
AI-infrastructure and software analysts tracking the data-cloud sector (DDOG, MDB, DBX).

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Micron crosses $1T market cap for the first time — UBS lifts PT to Street-high $1,625; MU +~19%</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Micron joins the trillion-dollar club as UBS triples its price target on sold-out HBM4 and structural AI-memory pricing.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Micron Technology shares jumped ~19% on May 26, pushing the memory chipmaker's market cap above $1T for the first time. The catalyst was a UBS note from analyst Timothy Arcuri lifting his price target to a new Street-high $1,625 from $535 — a 204% revision.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
UBS argued Micron's 2026 HBM4 capacity is sold out under long-term, fixed-price agreements, with DRAM contract prices up 58–63% and NAND up 70–75% YTD. The firm modeled EPS above $100 through 2029 even in a moderate downcycle, with cumulative FCF above $400B from 2027–2029.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Micron is now the 11th-largest U.S. public company by market cap, ahead of Walmart — marking memory's reclassification from commodity cycle to AI infrastructure. The Nasdaq closed at a fresh record 26,656 (+1.19%) led by the semis complex, and SanDisk, Western Digital, and Seagate all rallied in sympathy.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Semis traders and AI-infrastructure investors tracking the HBM supply chain.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Zscaler beat on Q3 and raised FY guides — then sliced its free-cash-flow margin outlook on AI capex, and the stock fell ~17–20% AH.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
