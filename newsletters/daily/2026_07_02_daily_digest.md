<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

If real, it converts Meta's headline capex (&gt;$100B/yr) from a cash drag into a hyperscaler-style revenue line.
</p>

AI-infrastructure traders, hyperscaler analysts, neocloud shorts.

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Bending Spoons (BSP) roars onto Nasdaq — IPO priced $29 above range, closes +39.66% at $40.50 for ~$26B implied market cap; AOL/Vimeo/Evernote owner raised $1.68B</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Italy's biggest consumer-software roll-up debuts on Nasdaq at $29, above range, and closes +40% — a firm signal the 2026 IPO window is still open.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Bending Spoons, the Milan-based owner of AOL, Vimeo, Evernote, WeTransfer, Brightcove, Eventbrite, Harvest, Remini, and StreamYard, priced its Nasdaq IPO at $29 per share on June 30 — above the $26–$28 marketed range — and began trading July 1 under ticker BSP.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Bending Spoons and selling shareholders sold 57.97M shares at $29, raising ~$1.68B in gross proceeds (Goldman Sachs, J.P. Morgan and Allen &amp; Company led). The stock opened $31, hit $43.98 intraday, and closed $40.50 (+39.66% vs IPO price) on 17.37M shares — pricing implied ~$18.4B, the debut close implied ~$25.7B.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Bending Spoons closing +40% from IPO is a very firm print for the 2026 new-issue market — it follows SpaceX's record June 12 IPO and validates that public bid for AI-adjacent and consumer-software names remains alive. The "buy-and-optimize" model — acquiring aging brands and re-monetizing them with a shared AI-heavy engineering stack — now has a public tape and multiples.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
IPO-flow traders and consumer-software allocators.

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Memory-stock rout — Micron (MU) -10.57%, SanDisk (SNDK) -10.62%, Western Digital (WDC) -6.32% as Q3 rebalancing and a California DRAM price-fixing suit weigh on sentiment</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">The 2026 AI-memory darlings take their sharpest hit of the year as a California DRAM lawsuit and Q3 rebalancing bite the winners.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Memory-chip stocks led a broad semiconductor pullback on July 1, the first trading day of Q3 2026. Micron (MU), SanDisk (SNDK), and Western Digital (WDC) — three of the year's biggest AI-driven winners — dropped 6% to 11% in a single session as institutional money took profits and a fresh price-fixing complaint hit the tape.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
MU closed $1,032.28 (-10.57%), SNDK closed $2,032.22 (-10.62%), WDC closed $598.37 (-6.32%). A California class action filed the prior week names Samsung, SK hynix and Micron as defendants, alleging they "illegally coordinated to restrict DRAM supply and inflate prices" — DRAM contract prices are up roughly 700% over four years.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Memory has been the single hottest AI-adjacent trade of 2026, with MU +305%, SNDK +858%, and WDC +271% YTD before Wednesday — a one-day 8–11% air-pocket on the first day of Q3 is what profit-taking looks like when the tape is that stretched. HBM and enterprise-NAND supply remains sold out through 2026 per Micron's June guide, so the structural bull case is intact.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Semi traders and AI-basket risk managers.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">US equities wrap Q2 2026 as best quarter since 2020 — Dow record 52,319.20, S&P 500 +14%, Nasdaq +20%; Russell 2000 +21% H1 (best since 1991)</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">US stocks close the best quarter since 2020 — Dow prints a second-straight record 52,319 as chips lead the final Q2 session.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Kroger (KR) to acquire Giant Eagle for $1.65B — $1.25B cash + ~$400M assumed liabilities buys 197 supermarkets and ~$9B in annual sales across OH, PA, WV, MD and IN</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Kroger returns to the M&amp;A table with a $1.65B tuck-in of family-owned Giant Eagle — a fraction of the failed $22B Albertsons deal, deliberately built to clear antitrust.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Kroger announced a definitive agreement Wednesday July 1 to acquire Giant Eagle, a family-owned regional grocer with roughly $9B in annual sales, for $1.65B — $1.25B cash and the assumption of ~$400M of Giant Eagle's outstanding liabilities. Giant Eagle operates 197 supermarkets and 11 standalone pharmacies across northern Ohio, western Pennsylvania, West Virginia, Maryland and Indiana.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Cash-financed transaction; Kroger keeps its $2B share repurchase program and dividend intact and expects to hold leverage at 2.3–2.5× net debt/EBITDA. Expected close is 2027 subject to HSR review, with "limited" Giant Eagle store divestitures anticipated to clear regulators. Deal is guided to be accretive to adjusted EPS in year two post-close.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
KR opened -2.8% pre-market on deal dilution and integration risk; it's the first meaningful test of the post-Albertsons playbook — buy regional, not national. Read-throughs: ACI, SFM and smaller regional grocers see the M&amp;A route re-open at a size the FTC will bless, and Ohio/western PA overlap creates a divestiture window for regional buyers.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Consumer-staples desks and grocery M&amp;A traders.
</p>

Warsh's refusal to endorse the AI-deflation frame removes a pillar of the summer-cut trade and reinforces the BofA "three-hikes-in-2026" path.

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">ADP June 2026 — private payrolls +98K vs +118K est misses; job-stayer pay holds at +4.4% Y/Y as natural resources sheds 5K jobs</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Private hiring cools 20K short of consensus as ADP prints +98K for June, but job-stayer wage growth stays sticky at +4.4% Y/Y.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The ADP National Employment Report showed US private-sector employment grew by 98,000 in June 2026, missing the Dow Jones / Reuters consensus of +118,000. It's a step down from May's +122K and a fourth straight sub-consensus month.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Education and health services led with +48K, followed by trade/transport/utilities +15K, financial activities +14K, manufacturing +5K. Natural resources and mining shed 5K jobs. Pay growth stayed sticky: job-stayers +4.4% Y/Y, job-switchers +6.6% Y/Y. The BLS NFP report for June follows Thursday July 2 (early close ahead of the July 3 Independence Day holiday).
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
A softer ADP print landing the same morning Warsh calls inflation "too high" tightens the setup for Thursday's NFP: the market wants a cooling-labor argument for the September FOMC path but not so cool that recession odds pop. Sticky pay growth is the tell — services desks watch stayer wages, not headline jobs, for the persistent piece of core CPI.
</p>

<strong style="color: #f7ff00;">Who is it for?</strong><br>
Macro traders, front-end rates desks, and jobs-print week readers.
