<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">U.S. consumer sentiment hits record low — U-Michigan May prelim 48.2 (lowest since 1952); gas prices and tariffs cited by ~63% of respondents</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">U-Michigan's May prelim consumer sentiment falls to 48.2 — the lowest reading in the survey's 74-year history.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The University of Michigan's preliminary May 2026 Index of Consumer Sentiment fell to 48.2 — down from April's 49.8 and below the 49.5 consensus — marking the lowest reading in the survey's 74-year history.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
About one-third of respondents spontaneously cited gasoline prices and ~30% cited tariffs as top concerns; one-year inflation expectations eased slightly to 4.5% from 4.7%.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Sub-50 sentiment prints have historically aligned with recession periods; with tariff overhang unresolved, follow-through into retail sales and PCE data would pressure consumer-discretionary names.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Macro traders, consumer and retail analysts watching the leading-indicator signal for discretionary spending.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">U.S. Court of International Trade rules Trump's 10% global Section 122 tariffs unlawful in 2-1 panel decision</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">CIT strikes down Trump's second-attempt 10% global tariffs as unlawful under Section 122 — but only the three named plaintiffs get relief.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
A three-judge CIT panel ruled 2-1 that Trump's February 2026 10% global import surcharge — imposed after IEEPA-based "Liberation Day" tariffs were struck down by the Supreme Court — was unlawful under Section 122 of the Trade Act of 1974.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The panel held the administration conflated balance-of-payments deficits with goods trade deficits; the permanent injunction covers only three named plaintiffs (Burlap &amp; Barrel, Basic Fun, Washington State) and the administration is expected to appeal to the Federal Circuit.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Both of Trump's two tariff legal vehicles have now been struck down; most importers still pay the 10% for now, but the precedent makes a broader injunction or class-wide refund more likely if the Federal Circuit appeal fails.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Macro traders, importers, retailers, and supply-chain analysts running tariff-scenario models for 2026.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">AWS US-East-1 thermal event takes Coinbase offline ~7 hours, hits FanDuel and CME — single-AZ outage cascades</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">An overheated data hall in northern Virginia took down a slice of the cloud — and the consumer apps that quietly run on it.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
On May 7, a thermal event in AWS's us-east-1 availability zone (use1-az4) knocked out EC2 instances and EBS volumes, cascading outages to Coinbase (~7 hours offline), FanDuel, and CME Group that extended into May 8.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Power was restored by 9:12 PM PDT on May 7, but ElastiCache, MSK (Kafka), and OpenSearch remained impaired for hours; Coinbase confirmed custody systems were unaffected while trading and withdrawals stayed locked.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
It is the second material AWS-driven Coinbase outage since October 2025, renewing scrutiny of single-region concentration risk for crypto exchanges, sportsbooks, and U.S. derivatives venues sharing the same Virginia footprint.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Infra and platform engineering teams, crypto traders, sportsbook operators, and exchange-risk desks evaluating multi-region redundancy.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Lime (Neutron Holdings) files S-1 for Nasdaq IPO under 'LIME' — flags going-concern risk on $846M debt due in 12 months</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Uber-backed micromobility platform files to go public with a going-concern warning attached.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Lime's parent Neutron Holdings filed an S-1 for an estimated $250M Nasdaq IPO under ticker LIME, led by Goldman Sachs, JPMorgan, Jefferies, and Evercore ISI.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
2025 revenue grew 29% to $886.7M with $104M of free cash flow, but $846M of debt matures within 12 months against only $261M in cash — the prospectus flags "substantial doubt" about going-concern unless the IPO or alternative financing closes.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
It is the first sizeable post-shakeout micromobility IPO; the going-concern flag makes pricing day a forced-sale refinancing event rather than a typical growth-IPO, with Uber — the largest outside investor — benefiting from a successful listing.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
IPO desks, micromobility and urban-transport watchers, and Uber analysts tracking the company's portfolio of platform bets.
</p>
