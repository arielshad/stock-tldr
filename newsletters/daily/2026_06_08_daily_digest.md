<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">OPEC+ Seven approve another 188,000 bpd quota hike for July — fourth consecutive monthly increase since Hormuz closure, cumulative unwind hits ~788,000 bpd</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Seven OPEC+ producers tack on another 188K bpd for July — a fourth straight 'symbolic' hike while Strait of Hormuz outages still cap real supply.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
At the 41st OPEC+ ministerial meeting in Vienna on June 7, seven members — Saudi Arabia, Russia, Iraq, Kuwait, Algeria, Kazakhstan and Oman — agreed to raise July collective output targets by 188,000 bpd, matching June's hike and marking the fourth consecutive monthly increase.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The April–July cumulative unwind now reaches ~788,000 bpd of the original 1.65M bpd voluntary cut, leaving ~376,000 bpd in place. Saudi Arabia and Russia each take roughly 62,000 bpd of the July increase; the UAE exited OPEC+ on May 1 and is no longer part of this tranche.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The paper hike is dwarfed by the ~10M bpd of Gulf supply still constrained by the Strait of Hormuz closure; Argus delegates framed it as 'preparing the ground' for real volumes once Hormuz reopens, not an immediate supply add. For energy desks the read is bearish-on-paper, neutral-in-reality.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Energy desks, oil traders, and macro overlay desks — watch for August JMMC commentary on the next 188K bpd tranche.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">May NFP +172K smashes the 80K bar and March/April get revised +93K higher — hawkish tape going into Warsh's first FOMC.</p>

The BLS Employment Situation for May 2026 showed nonfarm payrolls +172,000 — more than double the ~80K Dow Jones consensus — while the unemployment rate held at 4.3%, where it has stayed since July 2025.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
