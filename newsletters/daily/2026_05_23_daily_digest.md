<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

The proposed tie-up would have brought together Clinique, Tom Ford Beauty, and La Mer with Puig's Charlotte Tilbury, Jean Paul Gaultier, and Carolina Herrera into a ~$40B fragrance + skincare group.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

<strong style="color: #f7ff00;">Why does it matter?</strong><br>
EL jumped ~11% on the termination while Madrid-listed Puig fell ~14%. The collapse removes a key M&A catalyst from the prestige-beauty sector and re-focuses the EL bull case on its standalone 'Beauty Reimagined' turnaround under CEO de La Faverie.

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Consumer and luxury desks, M&A arb funds, and beauty cohort investors (Coty, L'Oréal, Shiseido).

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Japan's core CPI slowed to a four-year low of 1.4% in April — but the BOJ is still tracking toward a June hike.</p>

<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Core-core CPI (ex food and energy) fell to +1.9% from +2.4% the prior month. Energy prices dropped 3.9% Y/Y, while government cost-of-living subsidies and softer processed-food prices provided additional drags. Headline is now four straight months below the BOJ's 2% target.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Despite the downside miss, market consensus and Reuters reporting still place June as the likely venue for a 25bp BOJ hike to 1.00%. Governor Ueda appears to be treating the slowdown as transitory, keeping JPY traders on alert for any shift in communications.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
FX and macro traders, BOJ policy watchers, and JPY-sensitive equity desks.
