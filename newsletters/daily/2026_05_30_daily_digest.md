<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">STB pauses $85B Union Pacific–Norfolk Southern merger review — demands supplemental data by July 27; UNP -4.2%, NSC -4.9%</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Regulator hits pause on what would be America's first true transcontinental Class I rail merger.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The Surface Transportation Board unanimously accepted UP and NS's revised $85B all-stock merger application (Docket FD 36873) but immediately held proceedings in abeyance — ordering supplemental competitive and shipper-access data by July 27, 2026.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The STB found portions of the application "unclear or underdeveloped," requiring additional analysis on competition impact, gateway treatment, downstream M&A effects, and passenger-rail coexistence; the targeted close slips from April 2027 to mid-2027 as UNP fell 4.2% and NSC fell 4.9%.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The pause adds ~60 days to the regulatory clock and widens deal-arb spreads; most sell-side commentary keeps approval odds intact, but a longer review raises the risk of conditions that compress synergy upside on the largest US rail M&A ever.
</p>
