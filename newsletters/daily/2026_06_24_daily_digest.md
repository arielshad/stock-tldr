<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">KOSPI -9.99% — Twin Circuit Breakers as FSS Attacks Leveraged ETF Bubble</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Korea's chip kings tumble 12% as the FSS publicly attacks the leveraged ETF complex powering the rally.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
South Korea's <a href="https://finance.yahoo.com/quote/%5EKS11/" style="color: #f7ff00; text-decoration: none;">KOSPI</a> fell 9.99% to 8,203.84 on June 23 — a 910.71-point drop, the index's worst single session since March 4. <a href="https://finance.yahoo.com/quote/005930.KS/" style="color: #f7ff00; text-decoration: none;">Samsung Electronics</a> dropped 12.3% and <a href="https://finance.yahoo.com/quote/000660.KS/" style="color: #f7ff00; text-decoration: none;">SK Hynix</a> dropped 12.5%, the two stocks that make up nearly half the index, triggering twin circuit breakers.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The catalyst was a regulatory broadside: FSS Governor Lee Chan-jin publicly criticized single-stock leveraged ETFs tracking Samsung and SK Hynix, saying they had "done little more than enrich securities firms at retail investors' expense." AUM in those products swelled from KRW 4.5T to KRW 14T in under a month.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Foreign investors led the selling and the chip selloff spilled globally: Nasdaq 100 futures slid ~2.7% pre-open, <a href="https://finance.yahoo.com/quote/MU/" style="color: #f7ff00; text-decoration: none;">Micron</a> fell ~7.9% pre-market, and European semis dropped 5%+. This is the first big regulatory crackdown on the leveraged-vehicle plumbing of the AI chip trade.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Macro traders, semis investors, and anyone tracking global AI-chip contagion risk.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">AbbVie pays $10.9B cash for Apogee at $135.11/share, a 49.49% premium, to bolt-on a late-stage IL-13 immunology pipeline.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
<a href="https://news.abbvie.com/2026-06-22-AbbVie-to-Acquire-Apogee-Therapeutics,-Deepening-Immunology-Portfolio" style="color: #f7ff00; text-decoration: none;">AbbVie and Apogee Therapeutics</a> announced a definitive all-cash merger on June 22, 2026 at $135.11 per share — roughly $10.9B of equity value, a 49.49% premium to Apogee's Thursday close of $90.38. Both boards unanimously approved.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
It's AbbVie's largest bolt-on since Allergan and signals continued cash deployment into post-Humira immunology franchises. AbbVie says the deal becomes EPS-accretive in 2032 — a back-loaded payoff tied to APG777's ability to displace Dupixent in atopic dermatitis.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Chevron stands up a dedicated 2.67 GW gas-fired power complex in the Permian for a Microsoft AI data center next door.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Chevron and Microsoft announced <a href="https://www.worldoil.com/news/2026/6/22/chevron-signs-20-year-microsoft-power-deal-for-west-texas-ai-project/" style="color: #f7ff00; text-decoration: none;">"Project Kilby"</a> on June 22 — a co-located power complex and Microsoft data center in Reeves County, West Texas.
