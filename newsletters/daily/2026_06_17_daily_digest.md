The deal layers a $2.6B-ARR AI coding business onto SpaceX's launch/Starlink mix as hyperscalers race to own the developer surface — watch SPCX share count and the AI-coding peer set (<a href="https://finance.yahoo.com/quote/MSFT/" style="color: #f7ff00;">MSFT</a>/<a href="https://finance.yahoo.com/quote/GOOGL/" style="color: #f7ff00;">GOOGL</a>).

<a href="https://finance.yahoo.com/quote/AMD" style="color: #f7ff00;">Advanced Micro Devices</a> acquired MEXT, a memory-tiering startup whose predictive software lets NAND flash behave like DRAM — expanding the effective memory pool available to AI and HPC workloads. Financial terms were not disclosed; the MEXT engineering team joins AMD's data-center group.
</p>
<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
MEXT's software sits between the CPU/GPU and storage, keeping hot data pages in DRAM while transparently demoting cold pages to NAND flash — so applications see a far larger effective memory footprint at flash-tier cost. AMD plans to integrate it across EPYC server CPUs and Instinct GPUs.
</p>
<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Memory is the binding constraint on modern AI training and inference — this software layer stretches usable memory and chips away at <a href="https://finance.yahoo.com/quote/NVDA/" style="color: #f7ff00;">Nvidia</a>'s moat as AMD builds out its full data-center stack beyond silicon.
</p>
<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Semis investors, AI-infra desks, M&A trackers.
