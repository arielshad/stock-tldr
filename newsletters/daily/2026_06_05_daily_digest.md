<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">SpaceX nails the IPO price at $135/share before the roadshow even starts — pulling forward the most-watched listing in a generation.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
SpaceX filed an FWP with the SEC on June 3 fixing the IPO offering price at $135 per share — 555.6M shares to raise approximately $75B (the largest IPO ever) at a $1.75 trillion valuation. Roadshow begins June 4, pricing June 11, Nasdaq debut June 12 under ticker SPCX with up to ~30% retail allocation.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Posting a fixed price before the roadshow is highly unusual for US IPOs; Goldman Sachs, Morgan Stanley, BofA, Citi, and JPMorgan still handle the institutional bookbuild at that fixed $135. Musk retains over 82% voting control post-IPO via a multi-class structure; Q1 2026 revenue was $4.69B with adjusted EBITDA of $1.13B and Starlink crossing 10M subscribers.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
At $1.75T, SPCX would be the seventh-largest US-listed company at debut — instantly above Tesla. Index providers and ETF managers tracking S&P 500 and Nasdaq-100 will scramble for exposure; a successful $75B raise also drains meaningful liquidity from public-market positioning right as the Fed is on hold.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
IPO desks, indexers, retail flow desks, aerospace and satcom analysts.

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">USTR proposes Section 301 tariffs on 60 economies for forced-labor enforcement failures — 12.5% on China, Japan, South Korea, Brazil; 10% on EU, U.K., Canada, Mexico; public comments due July 6, hearings July 7</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">USTR floats 10–12.5% additional duties on 60 trading partners citing forced-labor enforcement failures.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
USTR Jamieson Greer's office published findings in 60 Section 301 investigations, declaring all 60 economies' enforcement of forced-labor import bans 'actionable' under the Trade Act of 1974 and proposing an additional-duty tariff overlay. This is the broadest single Section 301 action in recent memory.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Economies with reciprocal trade agreements (EU, U.K., Canada, Mexico) face a 10% additional duty; all others including China, Japan, South Korea, and Brazil face 12.5%. A textile mechanism allows reduced rates on certain apparel imports. Written comments are due July 6; hearings begin July 7.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
This is a parallel lever to the IEEPA-based reciprocal tariffs already in place — if implemented as proposed, it stacks on existing China duties for a combined effective rate well into the 30s. Read-through to apparel (NKE, RL, GAP, ANF), retailers with Asia sourcing (TGT, WMT, COST), and import-heavy industrials.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Trade-policy analysts, retail and import-supply-chain desks.
</p>
