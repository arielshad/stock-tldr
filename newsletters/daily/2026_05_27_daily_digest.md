<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Micron crosses $1T market cap as UBS lifts PT to Street-high $1,625 — MU +~19%</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Micron joins the trillion-dollar club as UBS triples its price target on sold-out HBM4 and structural AI-memory pricing.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Micron jumped ~19% to ~$895.88 on May 26, pushing its market cap above $1 trillion for the first time after UBS analyst Timothy Arcuri lifted his price target to a Street-high $1,625 from $535 — a 204% revision — citing sold-out HBM4 capacity and long-term AI-memory agreements.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
UBS modeled DRAM contract prices up 58–63% and NAND up 70–75% YTD, with cumulative FCF above $400B from 2027–2029 and EPS above $100 even in a moderate downcycle. Memory peers followed: SanDisk +~7%, Western Digital +~8%, Seagate +~4%.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Micron is now the 11th-largest US public company by market cap — ahead of Walmart — marking memory's reclassification from commodity cycle to AI infrastructure. The Nasdaq closed at a fresh record 26,656 (+1.19%) led by the semis bid.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Semis traders and AI-infrastructure investors watching the HBM/DRAM pricing cycle.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Zscaler Q3 FY26: beats on revenue and ARR, slashes FCF margin guide on AI capex — ZS -~17% AH</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Zscaler beat on Q3 and raised FY guides — then sliced its free-cash-flow margin outlook on AI capex, and the stock fell ~17–20% after hours.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
ZS's reaction is the latest SaaS-pays-the-AI-capex story — investors knocked ~$7B off market cap after hours despite raised top-line and EPS guides. Watch read-through to CRWD, NET, OKTA, S, and PANW on the same capex/margin tradeoff.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>

</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

Consumer discretionary investors and retail desks gauging DIY auto demand alongside peer reads from ORLY and AAP.
