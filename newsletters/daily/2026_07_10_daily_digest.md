<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Trump Kills the U.S.–Iran Ceasefire at NATO Ankara Summit</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Trump kills the 60-day U.S.–Iran ceasefire in Ankara — "they're scum" — as Treasury pulls Iran's oil waiver and CENTCOM strikes 80+ Iranian targets; WTI +5.2% to $74.16, Brent +6.1% to $78.66.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Speaking at the annual NATO summit in Ankara on July 8, President Trump declared that the ceasefire agreed with Iran in April is "over," following Iranian IRGC strikes on three Hormuz tankers and U.S. CENTCOM counter-strikes on 80+ Iranian military targets. Iran's IRGC responded by striking 85 U.S. positions in Bahrain and Kuwait.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Trump told reporters "I think it's over. I don't want to deal with them any more, they're scum"; the U.S. Treasury simultaneously revoked the sanctions waiver that had permitted Iran to sell crude under the June Bürgenstock MOU.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The MOU had underpinned a ~$10/bbl decline in crude since June 22; its collapse sent <a href="https://finance.yahoo.com/quote/CL=F" style="color: #f7ff00;">WTI +5.2% to $74.16</a>, Brent +6.1% to $78.66, VIX to 17.50 (+8.5%), and Dow futures down 468 points. Every $1/bbl above ~$72 tightens headline CPI arithmetic ahead of the July 14 print.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Macro traders, energy desks, and defensive-sector allocators watching the Middle East risk premium reprice in real time.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Micron (MU) +4.52% — Lifts US Chip Plan to $250B Through 2035, Pours First Clay NY Concrete</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Micron accelerates its US semiconductor investment plan to $250B+ through 2035 and marks the first concrete pour at its Clay, NY fab—more than one quarter ahead of schedule. SOX +4.1%, AMD +7.7%.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Micron raised its total planned US semiconductor investment to more than $250 billion through 2035 (up from $200B), coinciding with the first concrete pour at its Clay, NY memory fab—achieved more than one quarter ahead of the original schedule.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The plan funds up to four fabs at Clay, NY, plus Idaho expansions (first wafers mid-2027) and Virginia's active 1α DDR4 line, targeting 40% of Micron's DRAM output in the US and 90,000+ direct-plus-indirect jobs. A separate $3B commitment targets the domestic semiconductor supply chain.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
<a href="https://finance.yahoo.com/quote/MU" style="color: #f7ff00;">MU</a> closed +4.52% at $991.64 as the <a href="https://finance.yahoo.com/quote/%5ESOX" style="color: #f7ff00;">Philadelphia Semiconductor Index (SOX)</a> rallied 4.1% to 13,095.61 and <a href="https://finance.yahoo.com/quote/AMD" style="color: #f7ff00;">AMD</a> ripped +7.7%—cementing memory as a lead beneficiary of AI-era capex at a scale Micron's CEO called "foundational to the modern economy."
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Semis desks, memory-cycle traders, and US industrial-policy watchers tracking the HBM-sold-out-through-2026 thesis.

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Levi's beats top and bottom line, raises FY26 EPS and sales guide, and hikes the dividend 7%—but the stock still gaps down ~6% premarket as priced-in expectations rip past the raise.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Levi Strauss Q2 2026 (quarter ended May 31) posted adj EPS of $0.28 (beat $0.25 est) and net revenue of $1.56B (beat $1.52B est, +~8% Y/Y); the company raised FY26 EPS guide to $1.46–$1.52 (from $1.42–$1.48) and sales growth to +7–7.5% (from +5.5–6.5%).
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
