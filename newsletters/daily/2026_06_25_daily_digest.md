<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
After Tuesday's KOSPI rout dragged MU -13.2%, this print invalidates the "AI memory peaking" thesis — the Q4 EPS guide implies more profit in one quarter than the entire prior fiscal year. CEO Mehrotra disclosed 16 strategic supply agreements and confirmed HBM is fully booked through calendar 2026.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>

<strong style="color: #f7ff00;">What is it?</strong><br>
South Korea's KOSPI fell 9.99% to 8,203.84 on June 23 — a 910-point drop, the index's worst single session since March 4. Samsung Electronics dropped 12.3% and SK Hynix 12.5%, triggering two circuit breakers.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The catalyst is regulatory: FSS Governor Lee Chan-jin publicly criticized single-stock leveraged ETFs tracking Samsung and SK Hynix that launched in May, stating they had "enriched securities firms at retail investors' expense." AUM in those products swelled from KRW 4.5T at launch to KRW 14T in under a month.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

The selloff spilled globally: Nasdaq 100 futures slid ~2.7% pre-open, Micron fell ~7.9% in pre-market, and ASMI/STMicro dropped 5%+ in Europe. It is the first major regulatory crackdown on the leveraged-vehicle plumbing of the AI chip trade.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Macro traders and global semis investors tracking cross-border contagion risk from Korea's AI chip retail frenzy.
