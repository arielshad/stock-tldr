<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

Semiconductors led the rebound — SOXX +8.4% to $586.93, Intel +11.15% — with the session swallowing a simultaneously hot PPI print.

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Markets are pricing a Hormuz reopening path that would unwind the supply-side energy shock driving the inflation surge. The binary is whether Iran officially confirms — the naval blockade remains in force pending signing, and the Warsh FOMC on June 16–17 looms as the next catalyst.

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">U.S. May PPI +6.5% Y/Y — Hottest Wholesale Inflation Since Nov 2022 as Gasoline +23.4%</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Wholesale inflation runs hottest annual rate since Nov 2022 as Hormuz energy shock barrels through the supply chain.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
BLS released May 2026 PPI on June 11 — final demand +1.1% M/M (matching April) and +6.5% Y/Y, the highest 12-month gain since November 2022. Final-demand goods rose +2.8%, the largest one-month gain since records began in December 2009, led by energy +10.7% and gasoline +23.4%.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
About 80% of the goods surge traces to the +10.7% energy jump — gasoline alone accounted for more than half of the goods increase. Services rose only +0.3%. Core PPI (ex food, energy, trade) came in at +0.4% M/M / +4.9% Y/Y, both below consensus of +0.5% / +5.4%, providing a partial offset.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
PPI confirms the Hormuz supply-shock passthrough that already pushed May CPI to +4.2% Y/Y; goods-cost spikes typically feed consumer prices 2–3 months later, teeing up a hotter summer CPI run just as the Warsh FOMC meets June 16–17.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Macro traders, rates desks, and CPI watchers building pipeline-inflation models ahead of the June FOMC decision.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">May headline CPI hits a 3-year high of 4.2% as the Iran-driven energy shock filters through gasoline prices — core stays contained at 2.9%.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
