<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">SCOTUS splits the agencies — Fed carve-out survives 5-4, Humphrey's Executor overturned 6-3</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">SCOTUS protects the Fed (5-4) while gutting the rest of the independent-agency firewall (6-3) — Humphrey's Executor is dead, but Powell's board is not.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
On June 29 SCOTUS denied 5-4 Trump's stay in Trump v. Cook, keeping Fed Governor Lisa Cook in her seat, while ruling 6-3 in Trump v. Slaughter that the FTC's for-cause removal protection is unconstitutional, overturning the 1935 Humphrey's Executor precedent.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Chief Justice Roberts wrote both majorities — explicitly carving out the Federal Reserve from at-will removal while holding the FTC "unquestionably exercises executive power, and must therefore be controlled by the Chief Executive." Only Roberts and Kavanaugh sided with Cook in both cases.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Markets read the rulings as preserving Fed monetary-policy independence while opening FTC, SEC, NLRB, CFTC, and FCC commissioners to at-will removal; the S&P 500 closed +0.71% at 7,406.38 and the Nasdaq +1.29% at 25,623.67 on Fed carve-out relief.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Macro traders, rates desks, regulatory-policy PMs, and anyone trading agency-rule-sensitive sectors (financials, healthcare, tech).
</p>
