<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Mostly-stock offer for Dominion at a 21% premium would create the largest US power utility by far, aimed at AI data-center demand.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
NextEra would exchange ~0.8 of its shares per Dominion share, creating a combined enterprise value of ~$400B; NextEra shareholders would own ~75% of the pro-forma company.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Dominion's Virginia footprint sits atop Northern Virginia's "Data Center Alley" — the world's largest hyperscale cluster. Combining it with NextEra's renewables/gas fleet would lock in AI-buildout power supply; FERC and state PUC reviews would take 12–24 months.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Utility and AI infrastructure investors tracking power capacity for hyperscale data center buildouts — and anyone long NEE or D into a potential weekend announcement.
</p>
