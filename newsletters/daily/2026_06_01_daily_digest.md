<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">China May PMI — Manufacturing Slips to 50.0; Services Hold at 50.1</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">China's official factory gauge falls to the 50.0 expansion-contraction line in May — services hold up at 50.1, beating the 49.5 consensus.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The NBS released its May 2026 PMI prints: manufacturing PMI dropped 0.3 points to 50.0, matching consensus and sitting right at the expansion-contraction line. The non-manufacturing gauge slipped to 50.1 but beat the 49.5 estimate.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Inside the print, the production sub-index held at 51.2 but new orders dipped to 49.9 — output is firm but demand is softening. High-tech manufacturing PMI hit 52.9 for a 16th straight month above 50; large enterprises rose 0.9 points to 51.1.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
China is the swing input for the global manufacturing cycle — CSI 300, Hang Seng, copper, iron ore, and the offshore yuan all trade off these prints before Monday's Asia open. A 50.0 manufacturing read sustains the case for additional Beijing stimulus.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Macro traders, Asia desks, and commodity flow desks watching for Beijing's next policy move.
