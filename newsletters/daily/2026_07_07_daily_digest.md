<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Samsung Q2 2026: ₩89.4T (~$58.4B) operating profit — all-time company record</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Samsung's Q2 preliminary op profit lands at 89.4T won (~$58.4B), an all-time company record on AI memory demand.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Samsung Electronics released preliminary Q2 2026 earnings guidance on July 7 — consolidated sales of ~₩171T (+129.3% Y/Y) and operating profit of ~₩89.4T (~$58.4B), setting a new all-time corporate record.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
OP beat the ₩84.4T FnGuide consensus by ~6% and represents roughly 19x growth versus Q2 2025's ₩4.68T base, driven by HBM pricing and AI-memory shipment volume that also lifted Q1 2026 to ₩57.23T.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Confirms the AI-driven memory upcycle is still expanding, not plateauing. Direct read-through to Micron, Western Digital, TSMC and every HBM/DDR5 name in the supply chain.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
AI-memory and semis desks, Korea equity traders.

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">SK Hynix takes its Nasdaq ADR offering live — 177.9M ADS in what would be one of the largest foreign-issuer US debuts on record.</p>

<strong style="color: #f7ff00;">What is it?</strong><br>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Broadcom (AVGO) +4.3% — Apple partnership extended to 2031 with new custom ASIC</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Apple and Broadcom lock in five more years of custom chips and wireless connectivity, extending an ~$8B/yr revenue stream to 2031.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Broadcom announced on July 6 that it has extended its multi-year custom-silicon and wireless partnership with Apple through 2031, adding a new custom ASIC and renewing supply of Wi-Fi, Bluetooth, RF and networking chips.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The extension layers a new ASIC engagement on top of the 2023 RF agreement (runs to 2028). Apple represents ~20% of Broadcom's annual revenue; AVGO opened up 5%+, hit ~$381 intraday and settled +~4.3% at ~$376.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Kills the bear case that Apple's in-house modem work would phase Broadcom out of Wi-Fi/BT/RF — the 2031 lock-in pushes that risk out several years and validates Broadcom's "five-year annuity" narrative alongside the hyperscaler XPU ramp.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Semiconductor investors and Apple supply-chain traders.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">OPEC+ ratifies fifth straight 188,000 bpd quota hike for August — Brent ~$72</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">OPEC+ ratifies fifth straight monthly quota hike as Hormuz shipments start to recover.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Seven core OPEC+ producers — Saudi Arabia, Russia, Iraq, Kuwait, Algeria, Kazakhstan and Oman — agreed at a July 5 virtual meeting to lift collective output targets by 188,000 bpd starting in August, the fifth consecutive monthly increase.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Cumulative April–July quota additions now total ~800,000 bpd as the group unwinds a 1.65M bpd voluntary cut agreed in 2023; physical output fell from 42.77M bpd (Feb) to 33.13M bpd (May) during the Hormuz closure, so paper quotas have run well ahead of actual barrels.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Brent near $72 vs war-time peaks above $120 signals the group views Middle East supply capacity as largely restored; the measured 188K increment leaves room to slow if Iranian exports remain patchy or Iraq pushes for a bigger share.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Energy desks and macro traders.
</p>
