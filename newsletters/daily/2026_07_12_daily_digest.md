<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">SK Hynix (SKHY) Nasdaq debut at $149 — $26.5B raise breaks Alibaba's 12-year foreign IPO record, book ~7x oversubscribed; MU +4.5%, SNDK +7.6%, SMH +2.5%</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">SK Hynix lists on Nasdaq with a $26.5B raise — a new record for foreign IPOs and a fresh AI-memory anchor for US investors.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
SK Hynix — the world's #2 memory maker and Nvidia's main high-bandwidth-memory (HBM) supplier — priced its US Nasdaq IPO of American Depositary Shares at $149, raising ~$26.5B and eclipsing Alibaba's $25B 2014 debut for the largest-ever US IPO by a foreign company. SKHYV trades when-issued on July 10; regular-way ticker SKHY starts Monday July 13.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The offering covers 177.9M ADSs (each representing 1/10 of a Seoul common share), with books ~7x oversubscribed per Reuters sources — and Bloomberg reports it is the first US IPO priced at a premium to the home-market reference.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Direct HBM/DRAM exposure for US funds removes Korean withholding tax drag and triggers forced index buying (QQQ/NDX fast-track window), lifting memory-adjacent names (MU +4.5%, SNDK +7.6%, SMH +2.5%) and resetting the ceiling for future foreign mega-listings.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Semis desks, IPO/allocations teams, and Nvidia supply-chain analysts.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Circle (CRCL) +4.97% to $66.14 — OCC grants final approval for First National Digital Currency Bank; USDC issuer wins first federally supervised national trust charter for a US stablecoin firm</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Circle wins the first OCC national trust charter for a US stablecoin issuer, unlocking federally supervised USDC custody.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The OCC granted Circle final approval on July 10, 2026 to establish First National Digital Currency Bank, N.A. (dba Circle National Trust), converting the USDC issuer from a state-regulated stablecoin firm into a directly federally supervised national trust bank — joining Anchorage Digital as the only crypto companies with a national trust charter.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Circle originally filed the application June 30, 2025 and received conditional approval in December 2025; the trust will initially custody Circle affiliate assets, then expand to institutional customers such as banks, with USDC reserve management a designed future capability covering the $73.2B in circulation.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Federal OCC oversight replaces a state-regulated patchwork, materially reducing counterparty dependence on third-party custody banks; CRCL popped +15.6% intraday to $72.85 before fading to a +4.97% close at $66.14 as Mizuho flagged the move as 'overly optimistic' vs. rising Stripe/OpenUSD competition.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Stablecoin desks, crypto ETF flows watchers, and financial-regulation trackers following the race among Ripple, Paxos, BitGo, and Kraken for federal banking status.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The sale removes the strategic-vs-passive tension that has weighed on VOD since e&amp;'s 2022 stake buildup, triggering VOD ADR's largest single-day gain in years (+12.54% to $14.72); e&amp; books ~$1.28B in net return and redeploys capital to Gulf 5G and data-center infrastructure.
