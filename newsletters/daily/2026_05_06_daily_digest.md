<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Samsung Electronics tops $1T market cap as AI memory rally lifts shares ~12%; KOSPI jumps 5.4% to record</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Samsung joins TSMC in the trillion-dollar club as Korean memory cements itself at the heart of the AI buildout.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Samsung Electronics' market capitalization briefly crossed $1 trillion on Wednesday in Seoul as shares rallied as much as 12% on continued AI-chip momentum. It is the second Asian company ever to hit the threshold, after Taiwan Semiconductor.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The KOSPI benchmark jumped 5.4% to a record, with Samsung and SK Hynix together accounting for over 43% of the index. The rally extends a &gt;300% one-year run driven by record Q1 results: operating profit surged ~8x to 57.2T won and revenue hit a record 133.9T won as DRAM/NAND/HBM pricing kept climbing.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Confirms AI infrastructure demand has firmly broken the memory chip downcycle, reshaping global tech leadership. Spillover into US memory plays (MU, WDC), Asian fab equipment names, and Korean ETFs (EWY).
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
AI-infra and semis sector traders, Asian-equity allocators.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">U.S. Navy 'Project Freedom' Hormuz escort begins; Brent retraces ~4% to ~$110 as Iran ceasefire holds</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Day one of the U.S. Navy's Hormuz escort mission, with oil giving back most of the prior session's spike.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
After Iranian strikes on the UAE sent Brent up nearly 6% to $114, the U.S. operationalised 'Project Freedom' — a CENTCOM-led mission to escort merchant ships through the Strait of Hormuz with 15,000 personnel and multiple destroyer groups.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Brent settled at ~$109.87 (-4%) and WTI at $102.27 (-3.9%) as CENTCOM confirmed the U.S.-Iran ceasefire is intact. ~1,500 vessels remain stranded in the Gulf; only a handful have transited under escort so far.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Hormuz carries roughly a fifth of seaborne crude — any new strike on tankers or U.S. ships would re-price Brent fast. Energy names (XLE, OXY, XOM), refiners, airlines, and Treasuries are the primary read-throughs.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Energy desks and macro traders.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Apple reportedly in early talks with Intel and Samsung on US chipmaking; INTC closes ~+13%</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Bloomberg reports Apple is exploring Intel and Samsung as a US-side backstop to TSMC for main device chips.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Bloomberg reported Apple has held exploratory discussions with Intel and visited a Samsung plant in Texas about producing main device chips in the US — alongside, not replacing, longtime partner TSMC. No orders have been placed.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The motivation is supply diversification after AI data-center demand consumed leading-edge wafer capacity. Intel's restructured foundry and Samsung's Taylor, Texas facility are the candidate fabs; Apple has expressed concerns about reliability and scale outside TSMC.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Intel jumped ~12.9% to $108.15 on the implied option value of an Apple foundry win — 185.6M shares traded. Key confirm/deny gates: a concrete tape-out commitment, an Apple statement, or an 8-K from Intel or Samsung referencing a foundry agreement.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Semis, foundry, and large-cap tech investors.
</p>
