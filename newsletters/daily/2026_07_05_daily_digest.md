<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">BTC spot ETFs snap 10-day outflow streak with $221.7M net inflow Thursday as BTC rebounds from 21-month lows near $58K to ~$62.8K.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
